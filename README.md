Responsive Landing Page
As the lead web developer for the Ningo Prampram District Tech-hub, guide members of
the club with the following objectives:
 Practice selectors,
 layout
 fonts
 Media queries
Instructions/Guidelines:
You are to create a landing page for a farm products in the district using HTML & CSS only.
The page must include:
 A full-width hero section with background image
 A navigation bar fixed to the top
 Two-column section that becomes one column on screens < 700px
 Custom Google Font
 A button with hover and active states
Your focus skill sets include:
 Flexbox
 colors
 typography
 pseudo-classes
 responsive design
