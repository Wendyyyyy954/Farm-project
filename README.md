# Ningo Prampram Fresh Farms - Responsive Landing Page

 Project Overview
A professional, responsive landing page for a farm products business in Ningo Prampram District. Built with HTML & CSS featuring flexbox, responsive design, and custom Google Fonts.

 Features Implemented
✅ Fixed navigation bar with smooth scrolling
✅ Full-width hero section with gradient overlay
✅ Two-column product layout (responsive to single column <700px)
✅ 6 farm products with real images and pricing
✅ About section with company information
✅ Contact section with working form
✅ Professional footer with branding

 Technologies Used
- HTML5 (Semantic Structure)
- CSS3 (Advanced Styling)
- Google Fonts (Playfair Display & Poppins)
- Flexbox Layout
- Media Queries (Responsive Design)

 Skills Demonstrated
✅ CSS Selectors & Pseudo-classes
✅ Flexbox & Layout
✅ Typography & Color Theory
✅ Responsive Design (Mobile, Tablet, Desktop)
✅ Hover & Active Button States
✅ Media Queries (<700px breakpoint)
✅ Form Design
